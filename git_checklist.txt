[🚀 AutonomousVehiclePlatform - 작업 마무리 체크리스트]

✅ 1. GitHub와 동기화 여부 확인
---------------------------------
git pull

✅ 2. 작업 마무리 전 상태 확인
---------------------------------
git status

✅ 3. 변경사항 전부 스테이지에 올리기
---------------------------------
git add .

✅ 4. 커밋 메시지 작성 (명확하고 간단하게)
---------------------------------
git commit -m "기능 설명 또는 수정 요약"

✅ 5. GitHub에 반영
---------------------------------
git push

✅ 6. GitHub에서 정상 반영되었는지 확인
---------------------------------
- 올린 파일이 보이는지 확인
- README가 깨지지 않았는지 확인
